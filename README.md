# 🎬 IMDB Sentiment Analysis using NLP

## 📌 Overview

This project performs sentiment analysis on IMDB movie reviews to classify them as positive or negative. Natural Language Processing (NLP) techniques are used to preprocess text data and convert it into numerical features for machine learning models.

## 🧠 Model Used

* Machine Learning Classifier (e.g., Logistic Regression / Naive Bayes)

## ⚙️ Workflow

1. Data loading
2. Text preprocessing (cleaning, stopword removal, normalization)
3. Feature extraction using TF-IDF Vectorization
4. Train-test split
5. Model training
6. Prediction on test data
7. Model evaluation

## 🔤 Text Processing Techniques

* Lowercasing
* Removal of punctuation
* Stopword removal
* Tokenization

## 📊 Feature Engineering

* TF-IDF (Term Frequency–Inverse Document Frequency)

## 📊 Evaluation Metrics

* Accuracy Score
* Confusion Matrix (if applied)

## 📈 Result

The model successfully classifies movie reviews into positive and negative sentiments, demonstrating the effectiveness of TF-IDF with machine learning algorithms for text classification.

## 🚀 How to Run

* Open the notebook in Google Colab
* Run all cells sequentially

## 📁 Files

* `imdb-sentiment-analysis.ipynb`

* Basics of Natural Language Processing (NLP)
* Text preprocessing techniques

* Applying ML models to text data
